Hardware Implementation of Speech Synthesis System using FPGA

Problem Statement

  • speech synthesis system are software-based; thus, require the use of general-purpose computers for implementation

  • Difficult to embed on commercial products that require built-in speech synthesis system.

Project will

  • Receive digital signals as input and produce pre-recorded speech in real time to allow effective communication.
  • Implement speech synthesis on hardware to aid the visually impaired access visual information.
  • Provide an alternative, low-cost technology for a rapid translation.

    Aim
    To design and develop a feasible and cost-effective speech synthesis hardware system for the visually impaired.

    Objectives
  • To design and determine a speech synthesis algorithm for implementation on FPGA platform.
  • To develop a hardware prototype targeting the solution for the visually impaired.
  • To implement and test the system for real-time application
